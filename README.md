# 🎓 Lab Digital IPS - Pembelajaran Interaktif Kelas 7

Sistem laboratorium virtual digital untuk pembelajaran IPS (Ilmu Pengetahuan Sosial) kelas 7 yang dikembangkan berdasarkan **Draft Buku IPS Kelas 7**. Aplikasi ini menyediakan **12 lab interaktif** yang mencakup **SEMUA 4 TEMA** dalam buku, membuat pembelajaran IPS menjadi lebih menarik, visual, dan mudah dipahami.

## 📚 Fitur Utama

### 12 Laboratorium Virtual Interaktif - LENGKAP 4 TEMA!

#### TEMA I: Kehidupan Sosial dan Kondisi Lingkungan Sekitar (7 Lab) ✅

1. 🗺️ **Lab Geografi** - Lokasi Absolut, Relatif & Kondisi Geografis
   - Peta interaktif dengan OpenStreetMap/Leaflet
   - Bentuk muka bumi Indonesia (dataran rendah/tinggi/pegunungan/pesisir)
   - Pengaruh kondisi geografis terhadap mata pencaharian
   - Toponimi (studi penamaan tempat)

2. 🌍 **Lab Konektivitas** - Interaksi Antarwilayah
   - Simulasi visual konektivitas antarruang
   - Konsep "Saling Melengkapi" & "Kesempatan Berintervensi"

3. ☁️ **Lab Iklim** - Perubahan Iklim & Pencemaran Lingkungan
   - Tab Pencemaran (Udara, Air, Tanah) lengkap
   - Simulasi efek rumah kaca interaktif
   - Upaya pelestarian dengan checklist

4. 🌋 **Lab Bencana** - Bencana Alam & Mitigasi
   - Peta Indonesia interaktif dengan 3 lempeng tektonik
   - Marker gempa bersejarah & gunung berapi aktif
   - Panduan mitigasi lengkap

5. 👥 **Lab Sosial** - Interaksi & Dinamika Sosial
   - Dinamika sosial: Difusi, Asimilasi, Sosialisasi, Internalisasi
   - Culture shock & interaksi digital
   - Classification game interaktif

6. 🛒 **Lab Ekonomi** - Produksi, Distribusi, Konsumsi & Kelangkaan
   - Tab Kelangkaan dengan 3 masalah pokok ekonomi
   - Simulasi "Atur Anggaranmu" (Rp 100,000)

7. ⏰ **Lab Sejarah** - Timeline Geologis & Praaksara
   - Timeline interaktif 4 zaman geologis
   - Kjokkenmoddinger & evolusi praaksara

#### TEMA II: Keberagaman Lingkungan Sekitar (2 Lab) ✅

8. 🌱 **Lab SDGs** - Pembangunan Berkelanjutan
   - 4 Pilar SDGs interaktif
   - 17 Tujuan SDGs dengan visual grid
   - Aktivitas "Aksi Nyataku"

9. 🌳 **Lab SDA** - Sumber Daya Alam, Tambang & Kemaritiman
   - Klasifikasi SDA renewable/non-renewable (game)
   - Golongan tambang A, B, C
   - Tahapan pertambangan interaktif

#### TEMA III: Potensi Ekonomi Lingkungan (2 Lab) ✅

10. 🏪 **Lab Pasar** - Perdagangan & Mekanisme Pasar
    - Ekspor-impor & pelaku ekonomi
    - **Simulasi Supply-Demand** dengan slider harga
    - Visualisasi equilibrium

#### TEMA IV: Pemberdayaan Masyarakat (2 Lab) ✅ BARU!

11. 👑 **Lab Budaya** - Keragaman Budaya & Pemberdayaan Masyarakat
    - **Keragaman Sosial:** 1,300+ suku, 1,100+ bahasa
    - **Pahlawan Lokal:** Sultan Nuku, Ratu Kalinyamat, Malahayati
    - **Permasalahan Sosial:** Kesenjangan, kemiskinan, ketimpangan gender
    - **Peran Komunitas:** Pemberdayaan desa & komunitas
    - **Interaktif:** Jelajahi keragaman Indonesia per region
    - **Game:** Analisis masalah sosial

12. 💰 **Lab Keuangan** - Literasi Keuangan Keluarga
    - **Sejarah Uang:** Barter → Uang Kartal → Uang Giral
    - **Hukum Engel:** Hubungan pendapatan & konsumsi
    - **Tabungan vs Investasi:** Perbedaan & contoh
    - **Pengelolaan Keuangan:** Sistem Amplop, Buku Kas, Kas Keluarga
    - **Simulasi:** "Atur Keuanganmu" dengan budget Rp 2,000,000
    - **Literasi Digital:** Hindari pinjaman online & investasi bodong

## 🎯 Fitur Pembelajaran

### Setiap Lab Memiliki:
- **📖 Tab Materi** - Penjelasan lengkap dari buku dengan visual yang menarik
- **🎮 Tab Simulasi/Praktik** - Eksperimen dan simulasi interaktif
- **💡 Tab Kuis** - Quiz interaktif dengan feedback langsung

### Teknologi yang Digunakan:
- **React 18** dengan TypeScript untuk type safety
- **Vite** untuk development experience yang cepat
- **Tailwind CSS** untuk styling yang konsisten
- **Framer Motion** untuk animasi yang smooth
- **React Router** untuk navigasi antar halaman
- **React-Leaflet** untuk peta interaktif
- **Zustand** untuk state management
- **Lucide Icons** untuk ikon yang cantik

## 🚀 Cara Menjalankan

### Prerequisites
- Node.js 18+ 
- npm atau yarn

### Installation

1. **Install dependencies:**
```bash
npm install
```

2. **Run development server:**
```bash
npm run dev
```

Aplikasi akan berjalan di `http://localhost:5173`

3. **Build untuk production:**
```bash
npm run build
```

4. **Preview production build:**
```bash
npm run preview
```

## 🎮 Cara Menggunakan

### Untuk Siswa:
1. Buka aplikasi di browser
2. Pilih lab yang ingin dipelajari dari Dashboard
3. Baca materi di tab "Materi"
4. Coba simulasi interaktif di tab "Simulasi/Praktik"
5. Uji pemahaman di tab "Kuis"
6. Ulangi quiz untuk memperbaiki skor

- Gunakan sebagai media pembelajaran di kelas

- Gunakan quiz sebagai asesmen formatif
- Dashboard menunjukkan progress setiap lab
